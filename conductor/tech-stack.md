# Technology Stack

## Core Configuration
- **System Management:** Nix Flakes
- **macOS Configuration:** nix-darwin
- **User Environment:** Home Manager
- **Package Management:** Nixpkgs, Homebrew (via nix-homebrew)

## Languages
- **Configuration:** Nix
- **Scripting/Automation:** Lua (Hammerspoon), Bash/Zsh

## Tools & Applications
- **Shell:** Zsh
- **Editor:** Emacs
- **Secrets:** sops-nix
